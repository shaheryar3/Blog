# NCERT Reader Frontend

React + TypeScript frontend for a mobile-first NCERT reader.

## Scripts

In `/tmp/workspace/shaheryar3/Blog/frontend`:

- `npm start` — run development server
- `npm test -- --watchAll=false` — run tests once
- `npm run build` — create production build

## Configuration

Set `REACT_APP_NCERT_CATALOG_URL` to a public GitHub-hosted `catalog.json` URL.

If unset, the app uses a bundled sample catalog.
